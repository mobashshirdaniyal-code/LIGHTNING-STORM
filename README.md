# LIGHTNING-STORM
This is an interesting project for beginners using breadboard and LEDs in which different patterns will appear of lights will appear 
<br>
## COMPONENTS REQUIRED
<p>1. LEDs
2. 1 k-ohm Resister
3. Male to Male Jumper Wires
4. Arduino Uno
5. Breadboard
6. Power Resource</p>

# LIGHTNING-STORM
This is an interesting project for beginners using breadboard and LEDs in which different patterns will appear of lights will appear 
<br>
## COMPONENTS REQUIRED
<p>1. LEDs</p>
<p>2. 1 k-ohm Resister</p>
<p>3. Male to Male Jumper Wires</p>
<p>4. Arduino Uno</p>
<p>5. Breadboard</p>
<p>6. Power Resource</p>
